# Floway Tools

视频效果生成工具集，基于 WebGL + Canvas 2D 的浏览器端动画效果模板。

## 效果模板

| 模板 | 说明 |
|------|------|
| 图表效果 | 双风格动态折线图（纸张 + 赛博模式） |
| 柱状图效果 | 动态柱状图，支持纵向/横向 + 手绘抖动 |
| 饼图效果 | 动态饼图/环形图，阶梯缩放 |
| 文字动效 | 科技感标题动画引擎 |
| Logo 矩阵 | 循环 Logo 阵列背景 |
| 粒子场 | 动态粒子连线场 |
| 堆叠扫描 | 赛博朋克风格扫描文字 |
| 简洁折线图 | 简洁风格动态折线图 |
| 纸张折线图 | 纸张纹理风格折线图 |
| 3D 卡片 | 伪 3D 悬浮展示卡片 |

## 本地运行

```bash
# 启动本地服务器
./start.sh

# 或者用任意 HTTP 服务器
python3 -m http.server 8080
```

打开 `http://localhost:8000` 浏览效果。

## 技术栈

- Canvas 2D (2x supersampling)
- WebGL (3D 卡片合成)
- MP4/WebM/PNG 序列录制导出
- 纯前端，无框架依赖
